Spring 2020 Core seminars

In Fall 2020, Columbia College added 3 Core seminar sections that ended between 10am and 12pm GMT+8 time.

Number of seminars ending between 9pm and midnight also saw a slight increase by Spring 2021.

However, in comparison to the pre-COVID Spring 2020 semester, the total number of undergraduate level courses offered by Columbia saw a sharp decrease in Spring 2021.

Total number of undergraduate classes from 3050 to 2618
